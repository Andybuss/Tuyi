# Tuyi

图忆是一款基于位置信息，将时间空间结合的社交分享与个人事件记录的应用。主要实现了分享的事件在地图上直观呈现，打开地图就能发现附近分享的事件，附近分享搜索，附近用户雷达，个人时间轴记录。此外还有社交聊天，兴趣圈子等功能。
